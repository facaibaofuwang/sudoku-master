# Google Play 应用发布完整指南

## 📱 第一部分：创建 Google Play 开发者账号

### 1.1 注册开发者账号
1. 访问 https://play.google.com/console
2. 使用 Google 账号登录
3. 支付 $25 美元一次性注册费（信用卡/借记卡）
4. 填写开发者信息：
   - 开发者名称（显示在商店中）
   - 联系邮箱
   - 网站（可选）

### 1.2 验证身份
- 可能需要提供身份证或企业证件
- 等待审核通过（通常几分钟到几天）

---

## 📦 第二部分：创建应用

### 2.1 创建新应用
1. 登录 Google Play Console
2. 点击 "创建应用" 按钮
3. 填写基本信息：
   ```
   应用名称: 数独大师
   默认语言: 中文（简体）
   应用类型: 应用 或 游戏
   收费情况: 免费
   ```
4. 勾选声明，点击 "创建应用"

---

## 📤 第三部分：上传 APK

### 3.1 选择发布轨道
1. 左侧菜单：发布 → 测试 → 内部测试
2. 点击 "创建新版本"

### 3.2 上传 APK
1. 点击 "上传" 按钮
2. 选择文件：`/home/admin/openclaw/workspace/数独大师-v1.1.0-release.apk`
3. 等待上传和验证完成
4. 查看版本信息确认：
   - 版本代码: 2
   - 版本名称: 1.1.0
   - 包名: com.sudoku.master

### 3.3 版本说明
填写版本更新说明：
```
首次发布！
- 经典数独游戏
- 支持5种语言（中/英/俄/日/法）
- 多种难度选择
- 每日挑战
- 成就系统
```

### 3.4 保存并审核
1. 点击 "保存"
2. 点击 "审核版本"

## 🔒 第七部分：隐私政策

### 7.1 部署隐私政策
需要创建一个可访问的隐私政策页面。

#### 方案A：使用 GitHub Pages（免费）
1. 创建 GitHub 账号
2. 创建新仓库：`sudoku-master-privacy`
3. 创建 `index.html` 文件，内容使用 `PRIVACY_POLICY.md`
4. 启用 GitHub Pages
5. 获得链接：`https://yourname.github.io/sudoku-master-privacy/`

#### 方案B：使用免费托管服务
- Netlify
- Vercel
- Firebase Hosting

### 7.2 在 Play Console 添加链接
路径：政策 → 应用内容 → 隐私政策

填写链接：`https://your-domain.com/privacy-policy`

## 🔍 第九部分：应用审核

### 9.1 检查清单
提交前确认：
- [ ] APK 已上传
- [ ] 商店详情已填写
- [ ] 截图已上传
- [ ] 内容分级已完成
- [ ] 隐私政策已添加
- [ ] 定价已设置

### 9.2 提交审核
1. 左侧菜单：发布概览
2. 点击 "将更改送审"
3. 确认所有信息无误
4. 点击 "送审"

### 9.3 审核时间
- 新应用：通常 3-7 天
- 更新：通常几小时到 1-2 天

---

## 📊 第十部分：发布后管理

### 10.1 查看审核状态
- 发布概览页面查看审核进度
- 邮箱会收到审核结果通知

### 10.2 处理拒绝情况
如果审核被拒，常见原因：
- 隐私政策问题 → 更新隐私政策
- 应用崩溃 → 修复bug重新上传
- 内容违规 → 修改应用内容

### 10.3 正式发布
审核通过后：
1. 在 "发布概览" 页面
2. 点击 "开始发布"
3. 应用将在 Google Play 上架
