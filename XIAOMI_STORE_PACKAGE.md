## 🚀 六、上架操作步骤

### 步骤1: 注册小米开发者账号
1. 访问 https://dev.mi.com
2. 使用小米账号登录或注册
3. 完成实名认证（个人开发者需要身份证+人脸识别）

### 步骤2: 创建应用
1. 登录小米开放平台
2. 点击「应用发布」→「创建应用」
3. 选择分类：**游戏 > 益智休闲**
4. 填写应用名称：**数独大师**

### 步骤3: 填写应用信息
- 上传应用图标 (512x512 PNG)
- 上传应用截图 (3张，1080x1920)
- 填写应用介绍（使用上面的文案）
- 填写关键词（使用上面的关键词）

### 步骤4: 上传 APK
- 上传文件: `app-release.apk`
- 版本号: 2
- 版本名: 1.1.0

### 步骤5: 隐私合规
- 添加隐私政策 URL
- 选择适龄范围: 3+
- 确认无敏感权限

### 步骤6: 提交审核
- 检查所有信息
- 提交审核
- 等待审核结果 (1-3个工作日)

## 🌐 七、隐私政策部署指南

### 方案: GitHub Pages（免费）

#### 1. 创建 GitHub 账号
访问 https://github.com 注册

#### 2. 创建隐私政策仓库
```
仓库名: sudoku-master-privacy
可见性: Public
```

#### 3. 上传隐私政策文件
创建 `index.html`，内容如下：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>数独大师 - 隐私政策</title>
    <style>
        body { font-family: -apple-system, BlinkMacSystemFont, sans-serif; max-width: 800px; margin: 0 auto; padding: 20px; line-height: 1.6; }
        h1 { color: #333; border-bottom: 2px solid #4f46e5; padding-bottom: 10px; }
        h2 { color: #555; margin-top: 30px; }
        p { color: #666; }
        .date { color: #999; font-size: 14px; }
    </style>
</head>
<body>
    <h1>数独大师 隐私政策</h1>
    <p class="date">生效日期: 2024年1月1日 | 最后更新: 2026年2月12日</p>
    
    <h2>简介</h2>
    <p>数独大师（"我们"、"应用"）是一款经典的数独益智游戏。我们重视用户的隐私，本隐私政策旨在说明我们如何处理用户数据。</p>
    
    <h2>信息收集</h2>
    <p><strong>我们不收集的信息：</strong></p>
    <p>数独大师目前<strong>不收集</strong>任何个人身份信息，包括姓名、邮箱、电话号码、位置信息、设备唯一标识符等。</p>
    
    <h2>本地存储</h2>
    <p>应用仅在本地设备存储以下数据：</p>
    <ul>
        <li>游戏进度（保存的游戏状态）</li>
        <li>游戏统计（胜率、最佳时间等）</li>
        <li>成就解锁状态</li>
        <li>用户偏好设置（主题、声音）</li>
    </ul>
    <p>这些数据仅存储在您的设备上，不会上传到任何服务器。</p>
    
    <h2>第三方服务</h2>
    <p>当前版本不使用任何第三方分析或服务。</p>
    
    <h2>儿童隐私</h2>
    <p>本应用适合所有年龄段，包括儿童。我们不会故意收集13岁以下儿童的个人信息。</p>
    
    <h2>联系我们</h2>
    <p>如果您对本隐私政策有任何疑问，请联系我们：</p>
    <p>邮箱: [您的联系邮箱]</p>
</body>
</html>
```

#### 4. 启用 GitHub Pages
1. 进入仓库 Settings
2. 左侧选择 Pages
3. Source 选择 Deploy from a branch
4. Branch 选择 main / root
5. 保存后获得链接: `https://[username].github.io/sudoku-master-privacy/`

#### 5. 填入小米应用商店
将上述链接填入隐私政策 URL 字段。
