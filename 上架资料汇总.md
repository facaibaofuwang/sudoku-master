## 🚀 快速上架步骤

### 第1步：注册开发者账号（5分钟）
1. 访问 https://dev.mi.com
2. 使用小米账号登录
3. 完成实名认证

### 第2步：部署隐私政策（10分钟）
1. 注册 GitHub 账号
2. 创建仓库 `sudoku-master-privacy`
3. 上传 `privacy_policy.html` 作为 `index.html`
4. 启用 GitHub Pages
5. 记录 URL: `https://[username].github.io/sudoku-master-privacy/`

### 第3步：创建应用（10分钟）
1. 登录小米开放平台
2. 应用发布 → 创建应用
3. 选择分类: 游戏 > 益智休闲
4. 填写应用名称: 数独大师

### 第4步：填写资料（15分钟）
- 上传图标: `play_store_icon.png`
- 上传截图: 3张截图
- 填写介绍: 使用上方文案
- 填写关键词: 使用上方关键词
- 添加隐私政策 URL

### 第5步：上传 APK（5分钟）
- 上传: `app-release.apk`
- 版本号: 2
- 版本名: 1.1.0

### 第6步：提交审核（2分钟）
- 检查所有信息
- 提交审核
- 等待1-3个工作日
